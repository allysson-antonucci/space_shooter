# Space Shooter

A 2D top-down space shooter being developed with Godot.

## Current Features

- Player movement
- Multi-layer parallax background
- Shooting mechanics
- Enemies

## Planned Features

- Enemy wave system
- Boss battles
- User Interface (UI)

## Technologies

- Godot 4.3
- GDScript
