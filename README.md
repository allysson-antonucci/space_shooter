# Space Shooter

A 2D top-down space shooter being developed with Godot.

## Current Features

- Player movement
- Multi-layer parallax background

## Planned Features

- Shooting mechanics
- Enemies
- Score system
- Sound effects
- UI

## Technologies

- Godot 4.3
- GDScript
